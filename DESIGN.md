# Input controls

* Momentary push button (PB1)
* Mini push button (PB2)
* 6 potentiometers (POT*)
* 4 switches (SW*)
* 1 red led (LED1)
* 1 green led (LED2)

A possible usage would be:

  * Push buttons
    - PB1: Draw now
    - PB2: Change shape
  * Potentiomenters
    - POT1: Position X
    - POT2: Position Y
    - POT3: Angle
    - POT4: Size
    - POT5: Tempo (master clock) / Step size (external clock)
    - POT6: Pulse duration
  * Switches:
    - SW1: Manual draw
    - SW2: Clock: Master / external
    - SW3: Always on / Pulse
    - SW4: Alternate drawing cycle
  * Leds
    - LED1: Power
    - LED2: Drawing
